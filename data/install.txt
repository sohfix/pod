#### transcript
- **Transcribe MP3s** to text (English default; also French, Spanish, German, Italian).
- **Find words/phrases** and return **precise timestamps** with optional **buffer** before/after the hit.
- Cache a **word-level index** so repeated searches are instant.
- Output human-readable lines **or JSON** for scripting (e.g., auto-clipping with `ffmpeg`).

---

## Features

- **Transcription** powered by `faster-whisper` (fast, accurate, per-word timestamps).
- **Phrase search** across one file, many files, or entire directories.
- **Buffered timestamp ranges** (e.g., 3s pre-roll / 5s post-roll).
- **Cached word index** saved as `*.mp3.words.json` for instant re-search.
- **Persistent defaults** (e.g., default output directory).
- **Language options**: `english` (default), `french`, `spanish`, `german`, `italian`.

---
